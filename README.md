# Merge-sort
Sort a sequence of large numbers using merge sort. \

## Input

In the first line a number of integers N. In next N lines numbers to sort.


## Output

The sorted sequence in non-decreasing order, separated by new lines.


# Example

### Input:
```
5
13
27
328
111
25
```

### Output:
```
13
25
27
111
328
```
